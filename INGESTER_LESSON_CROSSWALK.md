# Ingester Lesson Crosswalk

This crosswalk maps repaired `ageo-atoms` lesson families to concrete `ageo-matcher` implementation and test coverage. It is intended as a planner-facing snapshot for deciding which matcher follow-up phase closes the highest remaining ingest-risk gaps without re-auditing both repos.

Use this as a routing document, not a source-of-truth spec. The implementation and test references below point to the current matcher evidence that was sufficient to verify whether each lesson is covered, partially covered, or still uncovered.

## Legend

- `covered`: matcher has direct implementation support plus focused tests or fixtures for the lesson.
- `partially_covered`: matcher has a narrow implementation slice, canary, or grouped example, but not broad enough coverage to treat the lesson as closed.
- `uncovered`: no meaningful matcher-side implementation or regression evidence was found for the lesson.
